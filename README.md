项目介绍:
1.在商品列表页面,所有商品只可以添加一次购物车,如果已经添加过了显示已经添加过购物车,如果没有则显示添加购物车,商品数量随着
2.在购物车页面可以对每一种商品进行数量的增减,但是最少不可以少于一件,总的商品数量和总的商品价格都会随之增加
